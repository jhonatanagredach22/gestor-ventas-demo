# Gestor de Ventas (DEMO)

Aplicación de escritorio desarrollada con **PHPDesktop** y **PHP nativo**, diseñada para gestionar ventas de forma local, simple y eficiente.  
Este proyecto demuestra una arquitectura limpia y modular basada en MVC.
